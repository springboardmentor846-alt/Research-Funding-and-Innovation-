# Phase 1 Implementation Scope

## Module 1: User Authentication and Role-Based Access

The system shall support:

1. User registration
2. User login using email and password
3. JWT-based authentication
4. OAuth2 login
5. Role-Based Access Control (RBAC)
6. Basic user profile management
7. Secure logout

### User Roles

- Researcher
- Startup Founder
- Innovation Manager
- Administrator

### Registration Rule

Public users may register as:

- Researcher
- Startup Founder
- Innovation Manager

Administrator accounts shall not be created through normal public registration.

---

## Module 2: Research Profile Management

The system shall support:

1. Research profile creation
2. Research profile viewing
3. Research profile updating
4. Research interest management
5. Research domain management
6. Keyword management
7. Publication management
8. Patent information management
9. Technology area management
10. Organization information management
11. Academic profile tracking
12. Research history management
